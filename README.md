# Substation Monitoring System
## About
Automated electrical substation monitoring  system built with python.

## What It Does
- Reads substation data from excel file
- Calculates Voltage and temprature
- Detects critical and warning faults
- Generates visual graph report

  ## Libraries Used
  - Numpy - calculations
  - Pandas - data analysis
  - Generates visual graph report
